# Resume-Analyzer
A simple resume analyzer that uses pre-defined keywords to extract skills from it.

<h3>Problem-Statement</h3>
It is inconvinient to extract skills from every resume by visually going over it and then noting the data. Then later checking it with each job role.

<h3>Working</h3>
This analyzer will extract skills from a resume that's provided to it and find the suitable job role for that skill set.
<h5>Input : </h5>
Resume text
<h5>Process : </h5>
i>  Extract skills using keywords.
ii> Compare with predefined job skill sets.
<h5>Output : </h5>
i>  List of the skills detected.
ii> Suggested job roles with match score.
